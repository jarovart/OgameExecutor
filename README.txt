The objective of this project is to start the expedition mission automatically. Build artifact e.g. with Intellij are required to execute.

1. Get the current version of your Google Chrome application "Help->About Google Chrome". E.g. Current version is 115.x
2. Get the current chromedriver from https://chromedriver.chromium.org/downloads, which matches to the version.
3. Replace it with chromedriver.exe on current project location.
